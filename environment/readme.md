# Miljö och installation – HW4
Detta dokument beskriver hur du installerar Python-paket och miljö för att köra projektet reproducerbart.

Se till att Jupyter notebook (gärna via Anaconda) samt Python-version 3.10 eller senare är installerat.  

Uppdatera pip
bash:
python -m pip install --upgrade pip

Installera pip
bash:
pip install -r requirements.txt
